## Setup with CocoPods
  - Add ```pod 'EMDAlertController'```  to your pod file.  
  - Run ```pod install```  
  - Run ```open App.workspace``` 
  
  You can then import the framework in your project.
  ```
  import EMDAlertController
  
  ```
